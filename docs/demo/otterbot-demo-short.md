# Otterbot Demo Script (Short)

## Scene 1: Introduction (10s)

Meet Otterbot — an open-source, multi-agent AI system that runs entirely on your machine.
You chat with a COO agent who manages teams of specialized workers, and you can watch every message flow in real-time.

## Scene 2: Getting Started (10s)

Install with one command, add your API key, and you're ready to go.
Otterbot supports Anthropic, OpenAI, Google Gemini, Ollama, and more — use any model you want.

## Scene 3: Delegation in Action (15s)

Tell the COO to "build me a portfolio website."
Watch it create a project, spawn a Team Lead, break the work into kanban tasks, and assign workers — all automatically.
The message stream shows every agent-to-agent conversation as it happens.

## Scene 4: Key Features (15s)

Fifteen specialized agents — coders, researchers, reviewers, browser automation, and more.
Integrated coding agent terminals with Claude Code, OpenCode, Codex, and Gemini CLI.
GitHub integration with automated issue triage, PR pipelines, and merge queues.
Voice support, 3D live view, Gmail, Calendar, and chat bridges for Discord, Slack, and Teams.

## Scene 5: Closing (10s)

Otterbot — your local AI team, fully transparent, fully yours.
Star the repo and get started today.
